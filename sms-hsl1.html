SMS-HSL: Simultaneous Multi-Slice Aliasing Separation Exploiting Hankel Subspace Learning Matlab Code
The following code includes a Matlab implementation of SMS-HSL for multiband brain MRI data to demonstrate some of the results that are decribed in the paper :
Suhyung Park, & Jaeseok Park. (2016), SMS-HSL : Simultaneous Mutislice Aliasing Separation Exploiting Hankel Subspace Learning. Magnetic Resonance in Medicine.



Note : The software is provided for non-commercial, and academic purpose only. Usage or distribution of the software for commercial purpose is forbidden. All rights belong to the related authors.  If you use the software for academic work, please cite the related publications.
